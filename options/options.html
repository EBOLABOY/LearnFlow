关于深学助手

了解应用的核心功能和隐私承诺

深学助手是一款专为在线学习优化的浏览器扩展，致力于提高学习效率，解放您的时间。通过智能识别课程页面结构，自动完成重复性的视频播放和章节测试任务。

🚀 核心功能

✓
智能识别学习平台页面结构
✓
自动播放视频课程，支持章节切换
✓
API智能拦截答题系统，实时获取题目答案
✓
人性化答错策略，模拟真实学习
✓
模块化架构，支持多平台扩展
✓
配置同步，跨设备使用
🔐

隐私承诺

所有配置和数据仅保存在您的浏览器本地，绝不上传至任何服务器。
